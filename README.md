# Proyecto C

## Descripción:
Este proyecto tiene el fin de seguir aprendiendo a diferenciar entre códigos de programación que necesitan de un compilador para ejecutarse.

## Prerrequisitos
* Instalar Visual Studio Code
* Instalar build-essential (compilador de C)


## Pasos para su ejecución:
1. Ejecutar Visual Studio Code.
2. Revisar que se tenga instalado apache2 con el comando "gcc -v".
3. En caso que no se tenga instalado alguno de los prerequisitos anteriores ejecutar los comandos "sudo apt install build-essential".
4. Buscar la carpeta "/var/www" en Visual Studio Code.
5. Dar permisos con el comando "sudo chown "ingresar su usuario":www-data html -R.
6. Ejecutar comando "sudo chmod 775 html -R".
7. Entrar en la carpeta html con el comando "cd html"
8. Crear carpeta "proyecto_c".
9. Descargar ambos archivos.
10. Copiar archivos en la carpeta "/var/www/html/proyecto_c".
11. Ingresar en la carpeta proyecto_c con el comando "cd proyecto_c".
12. Ingresar el comando "./conversor".
13. Utilizar el programa conversor de monedas.
